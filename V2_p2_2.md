# 2.2. Настройка среды разработки

Эта инструкция описывает шаги по настройке рабочего окружения на Windows для разработки проекта `FastMediaSorter_mobile` с использованием Android Studio и Visual Studio Code.

## Часть 1: Установка основных инструментов

### 1.1. Git
-   **Назначение:** Система контроля версий для работы с исходным кодом.
-   **Установка:**
    1.  Скачайте установщик с официального сайта: [git-scm.com](https://git-scm.com/).
    2.  Запустите установку. Во время установки рекомендуется оставить большинство настроек по умолчанию. Убедитесь, что Git добавлен в системный PATH, выбрав опцию "Git from the command line and also from 3rd-party software".

### 1.2. Android Studio
-   **Назначение:** Основная интегрированная среда разработки (IDE) для создания Android-приложений. Включает в себя Android SDK, эмулятор и инструменты сборки.
-   **Установка:**
    1.  Скачайте последнюю стабильную версию Android Studio с официального сайта: [developer.android.com/studio](https://developer.android.com/studio).
    2.  Запустите установщик и следуйте инструкциям. Убедитесь, что выбраны компоненты "Android Studio" и "Android Virtual Device".
    3.  После установки запустите Android Studio. Мастер первоначальной настройки поможет скачать последнюю версию Android SDK.

### 1.3. Visual Studio Code
-   **Назначение:** Легковесный редактор кода, удобный для работы с документацией (Markdown), скриптами и другими файлами, не связанными напрямую с компиляцией Android-приложения.
-   **Установка:**
    1.  Скачайте установщик с официального сайта: [code.visualstudio.com](https://code.visualstudio.com/).
    2.  Запустите установку. Рекомендуется включить опции "Add to PATH", а также "Add 'Open with Code' action" в контекстное меню.

## Часть 2: Клонирование репозитория
1.  Откройте терминал (Command Prompt, PowerShell или Git Bash).
2.  Перейдите в директорию, где вы планируете хранить проекты (например, `c:\GIT`).
3.  Выполните команду клонирования, используя URL-адрес, полученный на Шаге 2.1:
    ```bash
    git clone https://github.com/YourUsername/FastMediaSorter_mobile.git
    ```
4.  Перейдите в созданную директорию проекта:
    ```bash
    cd FastMediaSorter_mobile
    ```

## Часть 3: Настройка Android Studio
1.  **Открытие проекта:**
    -   Запустите Android Studio.
    -   На стартовом экране выберите **"Open"**.
    -   Перейдите к папке, куда вы клонировали репозиторий (`c:\GIT\FastMediaSorter_mobile`), и выберите ее.
2.  **Синхронизация Gradle:**
    -   Android Studio автоматически начнет процесс синхронизации проекта с использованием Gradle. Этот процесс может занять несколько минут, так как будут скачиваться все необходимые зависимости. Следите за прогрессом в нижней части окна IDE.
3.  **Настройка Android SDK:**
    -   Откройте SDK Manager (иконка куба со стрелкой вниз на панели инструментов или **Tools > SDK Manager**).
    -   Во вкладке **"SDK Platforms"** убедитесь, что установлены:
        -   Последняя стабильная версия Android (например, Android 13.0, API 33).
        -   **Android 9.0 (Pie, API 28)**, так как это минимальная требуемая версия для проекта.
    -   Перейдите во вкладку **"SDK Tools"** и убедитесь, что установлены:
        -   `Android SDK Build-Tools`
        -   `NDK (Side by side)`
        -   `Android SDK Platform-Tools`
        -   `Android Emulator`
4.  **Настройка устройства для запуска:**
    -   **Виртуальное устройство (Эмулятор):**
        1.  Откройте Device Manager (**Tools > Device Manager**).
        2.  Нажмите **"Create device"**.
        3.  Выберите модель устройства (например, Pixel 6) и нажмите "Next".
        4.  Скачайте и выберите образ системы (рекомендуется последняя стабильная версия API).
        5.  Завершите создание устройства, нажав "Finish".
    -   **Физическое устройство:**
        1.  На вашем Android-устройстве перейдите в **Настройки > О телефоне**.
        2.  Нажимайте на **"Номер сборки"** 7 раз, пока не появится сообщение "Вы стали разработчиком!".
        3.  Вернитесь в основное меню настроек, перейдите в **Система > Для разработчиков**.
        4.  Включите опцию **"Отладка по USB"**.
        5.  Подключите устройство к компьютеру с помощью USB-кабеля. В Android Studio оно появится в списке доступных устройств.

## Часть 4: Настройка Visual Studio Code
1.  Откройте VS Code.
2.  Выберите **File > Open Folder...** и укажите папку с проектом (`c:\GIT\FastMediaSorter_mobile`).
3.  Установите рекомендуемые расширения для удобной работы. Перейдите во вкладку **Extensions** (Ctrl+Shift+X) и установите:
    -   `EditorConfig for VS Code`: Автоматически применяет стили кода из файла `.editorconfig`.
    -   `GitLens`: Расширяет возможности по работе с Git внутри редактора.
    -   `Markdown All in One`: Все необходимое для редактирования Markdown-файлов (включая предпросмотр).
    -   `kotlin-language`: Базовая поддержка синтаксиса Kotlin.

## Часть 5: Первая сборка и запуск
1.  Убедитесь, что в Android Studio в выпадающем списке устройств выбрано ваше виртуальное или физическое устройство.
2.  Нажмите на зеленую кнопку **"Run 'app'"** (Shift+F10) на панели инструментов.
3.  Android Studio соберет проект и установит приложение на выбранное устройство. Первый запуск может занять значительное время.
4.  После успешной сборки приложение запустится на устройстве.
